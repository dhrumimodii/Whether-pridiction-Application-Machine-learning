#  Weather Prediction App

A simple Machine Learning app to predict **temperature** 🌡 and **rainfall** 🌧 using weather data.

##  Features

* Linear Regression for temperature prediction
* Random Forest for rain prediction
* Streamlit web interface

##  Run the App

```bash
streamlit run app.py
```

##  Inputs

* MinTemp
* MaxTemp
* WindGustSpeed
* Humidity
* Pressure

##  Output

* Predicted Temperature
* Rain (Yes/No)

## Tech Stack

Python, Pandas, NumPy, Scikit-learn, Streamlit

## Files
* app.py
* weather.csv
* temp_model.pkl
* rain_model.pkl
